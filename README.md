# 🃏 카드 뒤집기 기억력 게임 (Card Memory Game)

Python과 Streamlit을 이용해 만든 직관적이고 간단한 **초보자 수업용 starter 프로젝트**입니다. 

## 📌 프로젝트 소개
이 프로젝트는 Streamlit의 `session_state`를 직접 경험하며 상태 관리와 기본 UI 로직을 학습할 수 있도록 기획되었습니다. 복잡한 데이터베이스나 패턴 없이 3개의 핵심 파일만으로 16장(4x4)의 카드 짝 맞추기 기능을 완벽하게 구현했습니다.

## 🚀 실행 방법

1. **라이브러리 설치**
   아래 명령어를 통해 프로젝트에 필요한 패키지(`streamlit`)를 설치합니다.
   ```bash
   pip install -r requirements.txt
   ```
2. **애플리케이션 실행**
   아래 명령어를 통해 Streamlit 화면을 띄웁니다.
   ```bash
   streamlit run app.py
   ```
3. **브라우저에 접속**
   터미널에 안내된 **Network URL** (일반적으로 `http://localhost:8501`) 경로로 접속해 게임을 테스트해보세요.

## 📁 파일 구조

```text
card_game/
├── app.py             # Streamlit 화면 구성 및 메인 이벤트 실행 (UI)
├── game_logic.py      # 카드 섞기, 매치 확인 등 UI와 분리된 순수 게임 로직
├── state.py           # Streamlit의 상태 관리(session_state) 초기화 및 리셋 관리
├── requirements.txt   # 의존 패키지 선언
├── agent.md           # 본 프로젝트의 진행 규칙 및 가이드라인 문서
└── README.md          # 현재 보고 계신 프로젝트 소개 안내서
```

## ✨ 현재 구현된 핵심 기능
- **4x4 보드 게임판**: 총 8쌍(16장)의 과일 이모지 카드를 랜덤으로 섞어 배치합니다.
- **클릭 후 카드 뒤집기**: "❓" 뒷면인 카드를 클릭하면 해당 카드의 이모지가 나타납니다.
- **최대 두 장 뒤집기 비교**: 
  - 동시에 2장까지만 뒤집을 수 있습니다.
  - 선택한 2장이 **같으면** 그대로 앞면을 유지합니다.
  - **다르면** 사용자 확인을 위해 1초간 보여준 뒤 원래대로 ❓로 닫힙니다.
- **자동 게임 종료**: 16장을 전부 맞히면 성공 메시지와 함께 "다시 하기" 버튼이 표시되어 재도전할 수 있습니다.

## ❌ 제외 기능 (초기 버전 미구현)
초기 버전의 구조를 단순하게 유지하기 위해 아래 기능들은 구현하지 않았습니다:
- 타이머
- 점수 (시도 횟수 등)
- 최고 기록 저장
- 난이도 선택
- 힌트 제공
- 2인용 모드
